Sobre o Projeto

O Gestão Comunica Bem é uma iniciativa desenvolvida durante o Challenge, com o objetivo de facilitar os problemas enfrentados pela ong.

Nosso foco é criar um meio de comunicaçao e banco de dados facil e rapido em um lugar apenas

Objetivos principais

  • Possibilitar toda comunicaçao em apenas um local de maneira facil.
  • Armazenar os dados dos pacientes, disponibilizando formulario e envio de documenetos.
  • Disponibilizar os dados dos beneficiados para a ong.

Tecnologias utilizadas

HTML5, CSS3 (Flexbox e Grid), JavaScript e boas práticas de UX/UI.

Roadmap

Sprint 1: site estático - Sprint 2: formulários dinâmicos - Sprint 3: integração com banco de dados - Sprint 4: relatórios e gestão.
